# Database SQLite + interazioni

# Per eventuali grafici/report

# 📌 A cosa serve ognuno
# flask → il cuore del progetto (server web, routing, templates).

# python-dotenv → per leggere automaticamente il file .flaskenv con le impostazioni (es. FLASK_APP=app.py).

# sqlalchemy → utile se vogliamo gestire database SQLite/MySQL/Postgres in modo più “pulito” invece di scrivere query a mano.

# python-dateutil → gestire meglio le date (mesi, calcoli su giorni, ecc).

# matplotlib + pandas → se più avanti vogliamo generare report o grafici (es. incassi annuali).
